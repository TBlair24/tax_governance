URA Data Governance
Framework

An end-to-end data quality and governance pipeline for tax revenue data — detecting, scoring, and resolving data quality issues across six DAMA-aligned dimensions.

🚀 Launch Live App ⭐ View on GitHub

How the Pipeline Works

Seven components working together to take raw, messy tax data and produce clean, governed, auditable datasets.

🏭

Data Generation

Synthetic taxpayer register and tax return datasets with seven types of intentionally injected quality issues — mimicking real URA data problems.

🔍

Quality Checks Engine

Custom DQ engine assessing six DAMA-aligned dimensions — Completeness, Validity, Consistency, Uniqueness, Timeliness and Referential Integrity.

🔗

Lineage Tracking

SQLite-backed lineage catalogue recording every dataset, transformation and DQ run — creating a full audit trail from raw data to clean output.

🧹

Data Remediation

Automated cleaning pipeline that removes records failing quality checks — producing a clean, validated dataset ready for analytical use.

📊

Streamlit Dashboard

Interactive five-page dashboard with issue explorer, dimension scores, before vs after comparison, and full data lineage view.

📄

PDF Report Generator

Automated stakeholder report covering executive summary, dimension scores, issue log and governance policy — generated fresh on every pipeline run.
